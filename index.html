Presentación

Gracias a un vídeo tutorial que seguí en Youtube, he aprendido cómo hacer mi propio framework de CSS.

La idea general es ir añadiéndole cosas poco a poco y dejarlo en GitHub para su uso por otras personas. En perfecto castellano, para principiantes como yo. De forma, que finalmente, cada vez que necesite crear una página web, disponga de este código preparado.

El tutorial es de Carlos Arturo Esparza y podéis verlo pulsando en este enlace.

Recomiendo visitar su web : Falcon Masters

Otro gran divulgador al que le debo muchos de mis escasos conocimientos es JeanCarlos Molina Hostia de J&G Proyectos Web, del cual he copiado numerosos trucos de HTML y CSS.

Su canal de Youtube es estupendo para aprender de este mundillo. Pincha en este enlace para visitarlo.

Procuraré referenciar a toda la gente a la que le copio el código dentro del Framework así como en esta web.

No hago esto con fines lucrativos. Sólo para aprender.
